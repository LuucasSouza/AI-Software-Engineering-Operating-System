# Private Alpha Second Round Moderator Script

## Opening

This is a private alpha validation of Resolve Ai. The goal is to find confusion and safety issues before public release.

Do not use secrets, credentials, customer data, `.env`, dumps, backups, or confidential material.

## Flow

```text
resolve-ai ajuda
resolve-ai começar
resolve-ai ligar
resolve-ai diagnosticar
resolve-ai planejar
resolve-ai preparar
resolve-ai resolver
resolve-ai validar
resolve-ai status
```

## Required Questions

1. What does Resolve Ai do?
2. What does it not do?
3. Did `resolver` seem to execute changes automatically?
4. Did `executar` seem destructive?
5. Was the final agent prompt useful?
6. Did `status` make the next action clear?
7. Would you use it again?

## Closing

Thank the participant and record only anonymized, sanitized feedback.
