# Project Readiness Framework

## Objetivo

Tornar readiness observável por artefatos, quality gates, riscos e confiança explícita.

## Questions

1. Do we have required artifacts?
2. Are artifacts good enough for the next stage?
3. Are risks understood and acceptable?
4. Can the next agent continue without guessing?

## Status

Blocked, Weak, Acceptable, Strong and Excellent.
