# Blocking Conditions

## Examples

- No clear user or problem.
- MVP cannot be distinguished from future scope.
- Architecture has no alternatives.
- Decision has no criteria or consequences.
- High risk has no mitigation.
- Obvious simpler path ignored.
- Tasks lack acceptance criteria.
- Source of truth is unclear.
