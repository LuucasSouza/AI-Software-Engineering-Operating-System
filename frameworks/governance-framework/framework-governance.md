# Framework Governance

## Lifecycle

Draft, Experimental, Active, Stable, Deprecated and Archived.

## Metadata

Frameworks require title, version, status, owner, last_updated, review_cycle, related engines, protocols, templates and ADRs.

## Review Board Perspectives

Architecture coherence, product usefulness, engineering practicality, security/risk, documentation quality, agent executability, human reviewability and OSS maintainability.

## Deprecation

Deprecation requires reason, replacement, migration guidance, date, owner and affected documents.
