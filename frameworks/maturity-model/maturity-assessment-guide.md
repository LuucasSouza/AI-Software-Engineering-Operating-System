# Maturity Assessment Guide

## Scoring

| Score | Meaning |
| --- | --- |
| 0 | Not present |
| 1 | Present but informal |
| 2 | Documented and repeatable |
| 3 | Integrated with adjacent processes |
| 4 | Governed and reviewed |
| 5 | Continuously improved |

## Rule

Overall maturity is the minimum sustainable level across critical dimensions, not a simple average.

## Anti-Patterns

- Documentation theater.
- Process inflation.
- AI autopilot.
- ADR dumping.
- Enterprise cargo culting.
- Maturity vanity metrics.
