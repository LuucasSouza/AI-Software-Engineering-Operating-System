# AI-SEOS Maturity Model

## Objetivo

Medir a capacidade de aplicar AI-SEOS com clareza, rastreabilidade, governança e aprendizado.

## Levels

| Level | Name | Description |
| --- | --- | --- |
| M0 | Ad Hoc | Work is conversational and mostly undocumented. |
| M1 | Documented | Basic documents exist but consistency is limited. |
| M2 | Structured | Engines, templates and ADRs are used for important work. |
| M3 | Integrated | Engines are connected through handoffs and traceability. |
| M4 | Governed | Quality gates, reviews, ownership and versioning are enforced. |
| M5 | Adaptive | The system learns continuously through reflection loops. |

## Dimensions

Discovery, Product, Architecture, Decision, Risk, Optimization, Execution, Documentation, Handoff, Reflection, Agent Collaboration and Governance.
