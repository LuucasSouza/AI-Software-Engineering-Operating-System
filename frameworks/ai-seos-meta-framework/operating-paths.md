# Operating Paths

## Lightweight Path

Para experimentos, protótipos e baixo risco: Intake, Discovery Lite, MVP Scope, Architecture Lite, Decision Notes e Execution Plan.

## Standard Path

Para produtos profissionais: Intake, Discovery, PRD, Architecture, ADRs, Risk Register, Optimization Review, Execution Plan, Documentation, Handoff e Reflection.

## Enterprise Path

Para sistemas regulados, caros ou long-lived: Standard Path mais governance, compliance, security architecture, cost model, operational readiness, migration, rollback, review board e maturity assessment.
