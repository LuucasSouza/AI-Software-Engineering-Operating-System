# Reference Implementation Framework

## Objetivo

Provar a utilidade do AI-SEOS por projetos exemplo completos.

## Required Example Artifacts

Intake brief, discovery document, PRD, MVP scope, architecture overview, decision matrix, ADRs, risk register, optimization review, execution plan, documentation index, handoff package, retrospective and maturity assessment.

## Quality Rules

Reference implementations must be realistic, traceable, include trade-offs, show risks, rejected alternatives, scope reduction, architecture compromise and reflection improvement.
