# Agent Conflict Resolution

## Product vs Architecture

Create trade-off matrix, define reduced scope alternatives, escalate to AI CTO and produce decision record.

## Security vs Delivery

Classify risk severity, define compensating controls, document exception if accepted and require human approval for high risk.

## Optimization vs Maintainability

Quantify savings, quantify operational burden, compare reversal cost and decide through Decision Engine.
