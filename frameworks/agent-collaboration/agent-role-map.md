# Agent Role Map

| Agent | Primary Responsibility | Primary Engines |
| --- | --- | --- |
| AI CTO & Solution Architect | Technical strategy and framing | Core, Discovery, Architecture, Decision |
| Product Owner Agent | Product vision, PRD, MVP and backlog | Product |
| Principal Architect Agent | Architecture and trade-offs | Architecture, Decision, Risk |
| Security Architect Agent | Threats, security and compliance | Risk, Architecture |
| Implementation Lead Agent | Execution planning and sequencing | Execution |
| QA & Validation Agent | Acceptance and validation | Execution, Reflection |
| Documentation Agent | Documentation maintenance | Documentation |
| DevOps / Platform Agent | Deployment and operational readiness | Architecture, Execution, Risk |
| Reviewer Agent | Independent review | Decision, Risk, Reflection |
| Program Manager Agent | Milestones and delivery governance | Execution, Handoff |
