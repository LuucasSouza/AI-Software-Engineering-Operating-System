# Agent Collaboration Framework

## Objetivo

Governar colaboração multi-agent dentro do AI-SEOS.

## Filosofia

Agentes colaboram por artefatos estruturados, role clarity, human accountability e handoffs explícitos.

## Topologies

- Solo AI CTO.
- Specialist Chain.
- Review Board.
- Swarm with Coordinator.

## Human Review Required

Business-critical scope, irreversible architecture, security exceptions, compliance assumptions, high-cost infrastructure, production readiness, license/governance changes and public positioning.
