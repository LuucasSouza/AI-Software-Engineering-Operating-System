# Handoff Contract Model

## Objetivo

Definir obrigações do sender e do receiver.

## Sender obligations

- Provide truthful context.
- List artifact references.
- Make constraints explicit.
- Include known risks.
- Link decisions.
- Define done criteria.
- Surface open questions.

## Receiver obligations

- Validate understanding.
- Identify gaps.
- Follow constraints.
- Preserve traceability.
- Escalate when necessary.
- Produce expected outputs.
- Document deviations.
