# Kit simples de teste do Resolve Aí

Esta pasta e um kit simples para testar o Resolve Aí em outro projeto.

Ela nao e o Resolve Aí completo. Ela serve como guia para a pessoa que vai testar e para a IA que vai ajudar durante o teste.

## Como usar

1. Copie a pasta `teste/` para dentro do projeto que voce quer analisar.
2. Abra esse projeto no VS Code.
3. Abra o arquivo `COMECE_AQUI.md`.
4. Siga o passo a passo.
5. Durante o teste, preencha o arquivo `feedback.md`.
6. No final, envie o arquivo `teste/feedback.md` de volta para o Lucas.

## Arquivos principais

- `COMECE_AQUI.md`: comeca o teste, em linguagem simples.
- `PROMPT_PARA_IA.md`: prompt pronto para colar na IA dentro do projeto testado.
- `feedback.md`: formulario vivo que deve ser preenchido durante o teste.
- `SEGURANCA.md`: regras de seguranca e privacidade.
- `COMANDOS.md`: explicacao simples dos comandos do Resolve Aí.
- `EXEMPLO_FEEDBACK_PREENCHIDO.md`: exemplo ficticio de como preencher bem.
- `RESUMO_PARA_ENVIAR_AO_AMIGO.md`: mensagem pronta para convidar alguem para testar.

## Importante

Nao use um projeto com senha, token, `.env` real, backup de banco, dados de cliente ou informacao privada.

Se o projeto tiver dados sensiveis, use uma copia limpa ou uma pasta de teste.
