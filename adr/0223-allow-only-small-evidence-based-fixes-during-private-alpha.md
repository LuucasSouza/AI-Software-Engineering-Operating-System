# ADR 0223 - Allow Only Small Evidence-Based Fixes During Private Alpha

## Status

Accepted

## Context

Real feedback can trigger useful fixes, but scope expansion would destabilize alpha.

## Decision

Allow only small evidence-based fixes during private alpha.

## Consequences

- Copy, docs, and narrow UX bugs can be fixed.
- MCP, hooks, telemetry, and large commands remain out of scope.
- CLI fixes require tests and build.
