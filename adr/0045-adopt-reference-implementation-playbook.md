# ADR-0045 — Adopt Reference Implementation Playbook

## Status

Accepted

## Context

AI-SEOS needs examples that prove the framework can be applied end to end.

## Decision

Adopt a Reference Implementation Playbook and create an initial greenfield SaaS skeleton.

## Alternatives Considered

- Keep AI-SEOS only conceptual.
- Wait until all templates are complete before creating examples.

## Consequences

Future users can learn from concrete project structures.

## Risks

Skeleton examples may be mistaken for complete reference implementations.

## Reversal Strategy

Mark examples clearly by maturity and expand them incrementally.
