# ADR 0189 — Adopt Alpha Blocker Correction Backlog

## Status

Accepted

## Contexto

Blockers de release precisam de backlog separado de melhorias gerais.

## Decisão

Criar backlog específico para blockers/correções da v0.1.0-alpha.

## Consequências

Itens P1 antes de npm ficam claros e acionáveis.
