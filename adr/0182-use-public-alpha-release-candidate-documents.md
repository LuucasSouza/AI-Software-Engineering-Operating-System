# ADR 0182 — Use Public Alpha Release Candidate Documents

## Status

Accepted

## Contexto

Release prep precisa separar evidencia, notas, riscos, checklist e decisao.

## Decisão

Usar documentos de release candidate, release notes draft, checklist npm, risk review, security review e Go/No-Go antes de publicar.

## Alternativas consideradas

- Colocar tudo no README.
- Decidir apenas por score.
- Usar somente changelog.

## Consequências

Publicacao futura fica auditavel e menos dependente de memoria do mantenedor.
