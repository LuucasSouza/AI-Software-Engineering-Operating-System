# ADR 0175 — Adopt Release Recommendation Record

## Status

Accepted

## Contexto

Cada rodada alpha precisa terminar com decisao clara, nao apenas relatorio.

## Decisão

Registrar recomendacao proceed, hold ou extend apos cada rodada de validacao.

## Alternativas consideradas

- Deixar decisao no changelog.
- Discutir apenas informalmente.
- Decidir somente por score numerico.

## Consequências

O historico de release passa a separar evidencia, score, blockers e recomendacao.

## Plano de reversão

Uma politica de release futura pode incorporar este registro em um processo de aprovacao formal.
