# ADR 0217 - Require Real Participants Before Public npm GO

## Status

Accepted

## Context

Technical package readiness does not prove product readiness.

## Decision

Require at least one real participant per official profile before public npm GO.

## Consequences

- GO remains blocked with partial validation.
- Public release confidence improves.
- Profile-specific usability remains visible.
