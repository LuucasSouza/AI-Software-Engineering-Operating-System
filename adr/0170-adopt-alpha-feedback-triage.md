# ADR 0170 — Adopt Alpha Feedback Triage

## Status

Accepted

## Contexto

Feedback alpha precisa virar decisao sem misturar bugs, UX, seguranca e melhorias.

## Decisão

Adotar severidades P0, P1, P2 e P3 para triagem de feedback alpha.

## Alternativas consideradas

- Usar labels informais.
- Tratar feedback por ordem de chegada.
- Priorizar apenas bugs tecnicos.

## Consequências

P0 bloqueia release e P1 deve ser resolvido antes de publicacao npm ampla.

## Plano de reversão

O modelo pode ser substituido por processo de issue triage mais completo.
