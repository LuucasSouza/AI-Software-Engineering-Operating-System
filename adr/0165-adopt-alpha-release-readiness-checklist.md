# ADR 0165 — Adopt Alpha Release Readiness Checklist

## Status

Accepted

## Contexto

Release alpha precisa de critérios explícitos além de testes passando.

## Decisão

Adotar checklist de readiness para `v0.1.0-alpha`.

## Alternativas consideradas

- Publicar com base apenas em testes.
- Depender de revisão informal.
- Adiar qualquer checklist.

## Consequências

O release fica mais previsível e honesto sobre limitações.

## Plano de reversão

Substituir por processo de release mais formal em versões posteriores.
