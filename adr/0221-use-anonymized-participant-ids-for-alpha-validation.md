# ADR 0221 - Use Anonymized Participant IDs for Alpha Validation

## Status

Accepted

## Context

Real alpha validation involves people and potentially real projects.

## Decision

Use anonymized IDs such as `P15B-VC-01`.

## Consequences

- Reduces privacy risk.
- Preserves enough traceability for product decisions.
