# ADR 0219 - Treat Resolver/Executar Perception as P1 Validation Item

## Status

Accepted

## Context

`resolver` and `executar` can sound like automatic execution.

## Decision

Treat relevant confusion about these commands as P1 for public release.

## Consequences

- Safety copy must stay explicit.
- Public GO requires correct user perception.
- Tests and sessions must check this.
