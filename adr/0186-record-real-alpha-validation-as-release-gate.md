# ADR 0186 — Record Real Alpha Validation as Release Gate

## Status

Accepted

## Contexto

Public alpha não deve depender apenas de testes simulados.

## Decisão

Registrar validação real por perfil como gate explícito antes de GO público.

## Consequências

Sem usuários reais, a decisão máxima permanece LIMITED GO.
