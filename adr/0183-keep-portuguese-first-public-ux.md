# ADR 0183 — Keep Portuguese-First Public UX

## Status

Accepted

## Contexto

Resolve Ai tem promessa publica em portugues brasileiro.

## Decisão

Manter experiencia publica, quickstart e comandos com linguagem brasileira clara.

## Alternativas consideradas

- Migrar docs principais para ingles.
- Usar tom enterprise tecnico.
- Deixar UX para depois.

## Consequências

O projeto continua acessivel a Non-Technical Builders e Vibe Coders no Brasil.
