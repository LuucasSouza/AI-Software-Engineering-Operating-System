# ADR-0048 — Adopt Vibe Coder Mode

## Status

Accepted

## Context

Many users build with AI tools and need practical guidance, sequencing and guardrails.

## Decision

Adopt Vibe Coder Mode as an official entry mode for AI-assisted builders who need controlled implementation phases.

## Alternatives Considered

- Treat vibe coders as non-technical users.
- Treat vibe coders as professional engineers.

## Consequences

AI-SEOS can produce Codex-ready prompts, validation checkpoints and safer implementation sequencing.

## Risks

Users may still skip guardrails to move faster.

## Reversal Strategy

Merge Vibe Coder Mode into Execution templates if the mode becomes redundant.
