# ADR 0173 — Adopt GitHub Alpha Issue Templates

## Status

Accepted

## Contexto

Feedback externo tende a chegar incompleto sem estrutura.

## Decisão

Criar issue templates para alpha feedback, bugs de CLI, confusao de UX, docs e preocupacoes de seguranca.

## Alternativas consideradas

- Usar apenas issue em branco.
- Coletar feedback fora do GitHub.
- Criar somente um template generico.

## Consequências

Feedback fica mais facil de triar e menos propenso a expor dados sensiveis.

## Plano de reversão

Templates podem ser consolidados se criarem atrito excessivo.
