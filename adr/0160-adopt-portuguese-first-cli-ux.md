# ADR 0160 — Adopt Portuguese-First CLI UX

## Status

Accepted

## Contexto

Resolve Aí é uma experiência pública em português.

## Decisão

Manter comandos e mensagens públicas em português brasileiro, com aliases sem acento quando necessário.

## Alternativas consideradas

- Migrar para inglês.
- Misturar comandos em inglês e português.
- Usar apenas comandos técnicos.

## Consequências

A CLI permanece acessível para usuários brasileiros e coerente com a marca.

## Plano de reversão

Adicionar internacionalização futura sem remover a experiência pt-BR.
