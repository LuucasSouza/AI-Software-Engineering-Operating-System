# ADR 0086 - Adopt Runtime Direction Inspired by Ease of Use

## Status

Accepted

## Context

Resolve Aí should become easier to activate inside projects during Phase 3.

## Decision

Use ease-of-use patterns such as CLI commands, adapters, project-local state, agent instructions and future MCP tools, without copying Context Mode.

## Consequences

Phase 3 can focus on software engineering orchestration with a clear and approachable runtime identity.
