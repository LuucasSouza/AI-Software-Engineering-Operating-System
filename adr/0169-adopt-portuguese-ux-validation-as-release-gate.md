# ADR 0169 — Adopt Portuguese UX Validation as Release Gate

## Status

Accepted

## Contexto

A experiencia publica principal do Resolve Ai e em portugues.

## Decisão

Tratar clareza de comandos, mensagens e documentos em portugues como gate de release alpha.

## Alternativas consideradas

- Validar apenas funcionalidade.
- Priorizar ingles primeiro.
- Corrigir linguagem depois do npm.

## Consequências

Feedback de usuarios brasileiros passa a influenciar readiness e backlog.

## Plano de reversão

Uma futura estrategia multi-idioma pode adicionar outros gates, sem remover o portugues-first.
