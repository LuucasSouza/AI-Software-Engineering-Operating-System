# ADR 0179 — Require Real User Validation Evidence

## Status

Accepted

## Contexto

Validacao simulada nao prova entendimento humano nem utilidade real.

## Decisão

Public alpha npm exige evidencia de validacao real ou decisao explicita de alpha controlado.

## Alternativas consideradas

- Aceitar apenas testes automatizados.
- Inventar validacao com base em simulacao.
- Publicar e coletar feedback depois.

## Consequências

Sem validacao real, a recomendacao maxima e LIMITED GO para alpha controlada.
