# ADR 0174 — Adopt Private/Controlled Alpha Before Public Distribution

## Status

Accepted

## Contexto

Publicacao ampla antes de validar valor pode gerar feedback ruidoso e regressao de confianca.

## Decisão

Executar alpha controlada antes de distribuicao publica ampla.

## Alternativas consideradas

- Publicar direto em npm.
- Fazer apenas validacao simulada.
- Adiar qualquer usuario externo.

## Consequências

O projeto prioriza aprendizado controlado antes de alcance.

## Plano de reversão

Uma rodada futura pode liberar public alpha se nao houver P0 e P1 criticos.
