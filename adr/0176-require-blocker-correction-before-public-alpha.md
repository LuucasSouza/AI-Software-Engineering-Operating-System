# ADR 0176 — Require Blocker Correction Before Public Alpha

## Status

Accepted

## Contexto

A Phase 11 identificou P1s e P2s que reduziam confianca para public alpha.

## Decisão

Nenhum public alpha deve ser recomendado enquanto P1s conhecidos estiverem sem correcao, mitigacao ou decisao explicita documentada.

## Alternativas consideradas

- Publicar com blockers conhecidos.
- Tratar blockers apenas no changelog.
- Adiar qualquer release prep.

## Consequências

Release passa a depender de evidencias e decisoes registradas.
