# ADR 0224 - Avoid Infinite Documentation-Only Validation Loops

## Status

Accepted

## Context

Repeated phases without participants can create false progress.

## Decision

Avoid infinite documentation-only validation loops. If validation is blocked, the main next action must be participant recruitment.

## Consequences

- Roadmap becomes more operational.
- Documentation does not replace user evidence.
