# ADR 0216 - Adopt Validation-Blocked State for Private Alpha

## Status

Accepted

## Context

Previous phases risked creating documentation without real participants.

## Decision

Adopt `VALIDATION BLOCKED` as a formal state when no real participants are available.

## Consequences

- Prevents readiness inflation.
- Makes operational recruitment explicit.
- Avoids documentation-only loops.
