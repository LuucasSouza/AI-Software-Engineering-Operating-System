# ADR 0206 - Adopt Private Alpha Expansion Before Public Release

## Status

Accepted

## Context

Phase 14 ended with LIMITED GO because real user validation was still missing.

## Decision

Expand private alpha before public npm release.

## Consequences

- Public release remains delayed.
- Product risk is reduced before wider distribution.
- Private feedback becomes the next release gate.

## Alternatives Considered

- Publish npm based only on technical readiness.
- Continue producing documentation without participant sessions.
