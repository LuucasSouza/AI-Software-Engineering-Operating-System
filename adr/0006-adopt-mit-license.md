# ADR-0006 - Adopt MIT License

## Status

Accepted

## Contexto

O AI-SEOS será publicado como framework open source composto principalmente por documentação, templates, protocolos, agentes e exemplos.

## Problema

O repositório precisa de uma licença clara para permitir uso, cópia, modificação, distribuição e contribuição com baixa fricção.

## Opções consideradas

- MIT License.
- Apache License 2.0.
- Creative Commons para documentação.
- Licença a definir posteriormente.

## Decisão

Adotar MIT License como licença inicial do projeto.

## Consequências positivas

- Baixa fricção para adoção.
- Simplicidade para contribuidores.
- Compatibilidade ampla com projetos open source.

## Consequências negativas

- Não oferece proteções explícitas de patente como Apache 2.0.
- Pode ser menos específica para conteúdo documental do que licenças Creative Commons.

## Trade-offs

Ganha-se simplicidade e adoção ampla. Aceita-se menor especificidade para documentação e patentes.

## Plano de reversão

Se o projeto exigir proteção adicional, criar ADR futura propondo Apache 2.0 ou modelo dual-license, com plano de migração para contribuições futuras.
