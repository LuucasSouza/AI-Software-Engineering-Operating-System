# ADR 0194 — Require Maintainer Release Approval

## Status

Accepted

## Contexto

Publicação npm e tag são ações públicas e manuais.

## Decisão

Exigir aprovação explícita do mantenedor antes de qualquer npm publish, tag ou release GitHub.

## Consequências

Automação ou execução acidental não vira release público.
