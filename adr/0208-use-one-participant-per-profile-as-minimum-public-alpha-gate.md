# ADR 0208 - Use One Participant Per Profile as Minimum Public Alpha Gate

## Status

Accepted

## Context

Resolve Ai serves Non-Technical Builder, Vibe Coder, and Professional Engineer profiles.

## Decision

Use one real participant per official profile as the minimum public alpha gate.

## Consequences

- Each target audience must be represented.
- Gaps in one profile block GO.
- Feedback can be evaluated per profile instead of averaged away.

## Alternatives Considered

- Validate only with technical users.
- Validate only with a single general participant.
