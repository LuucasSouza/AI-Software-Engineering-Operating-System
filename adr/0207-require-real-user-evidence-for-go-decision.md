# ADR 0207 - Require Real User Evidence for GO Decision

## Status

Accepted

## Context

Technical tests cannot prove user clarity, trust, or safe interpretation.

## Decision

Require real user evidence before deciding GO for public alpha.

## Consequences

- GO cannot be based on templates or simulated runs alone.
- Missing participants keep the decision at LIMITED GO or lower.
- Release confidence improves.

## Alternatives Considered

- Treat local smoke tests as sufficient.
- Use maintainer judgment without participant evidence.
