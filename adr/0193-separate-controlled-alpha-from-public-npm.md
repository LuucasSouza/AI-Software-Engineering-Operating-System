# ADR 0193 — Separate Controlled Alpha From Public npm

## Status

Accepted

## Contexto

Usuários convidados podem validar localmente sem publicação pública.

## Decisão

Separar alpha controlada de public npm release.

## Consequências

Feedback real pode ser coletado antes de ampliar distribuição.
