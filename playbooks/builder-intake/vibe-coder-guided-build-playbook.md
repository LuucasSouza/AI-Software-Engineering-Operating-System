# Vibe Coder Guided Build Playbook

## Goal

Turn an idea into safe, sequential AI-assisted implementation phases.

## Steps

1. Run small discovery.
2. Define MVP boundary.
3. Choose simple architecture with rationale.
4. Split implementation into phases.
5. Write one Codex-ready prompt per phase.
6. Validate after every phase.
7. Commit between phases.
8. Run refactor and risk pass before deploy.

## Guardrail

Never generate the entire app in one prompt.
