# Reference Implementation Playbook

## Steps

1. Select project type.
2. Create intake brief.
3. Run Discovery.
4. Generate Product Definition.
5. Generate Architecture.
6. Run Decision Review.
7. Run Risk Assessment.
8. Run Optimization.
9. Build Execution Plan.
10. Produce Documentation and Handoff.
11. Reflect and assess maturity.
