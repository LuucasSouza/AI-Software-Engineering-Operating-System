# Playbooks

## Objetivo

Documentar guias práticos para cenários recorrentes de engenharia orientada por IA.

## Escopo

Inclui greenfield SaaS, modernização de legado, MVP de startup, plataformas enterprise, produtos AI-first e sistemas regulados.

## Princípios

- Playbooks devem orientar ação.
- Decisões importantes devem apontar para ADRs ou matrizes.
- Cada playbook deve indicar riscos e limites.

## Responsabilidades

Playbook Maintainers mantêm aplicabilidade prática e alinhamento com templates e protocolos.
