# Code of Conduct

## Objetivo

Estabelecer padrões mínimos de colaboração para manter o AI-SEOS aberto, técnico, respeitoso e produtivo.

## Escopo

Aplica-se a contribuidores humanos, maintainers, revisores e agentes de IA operando neste repositório ou em espaços oficiais do projeto.

## Princípios

- Respeito técnico e pessoal.
- Crítica focada em artefatos, decisões e trade-offs.
- Transparência sobre uso de IA.
- Abertura para contribuição responsável.
- Responsabilidade humana final em decisões sensíveis.

## Comportamentos esperados

- Explicar decisões com clareza.
- Registrar mudanças estruturais em ADRs.
- Respeitar escopo e governança.
- Tratar discordâncias como análise de trade-offs.
- Corrigir erros de forma rastreável.

## Comportamentos inaceitáveis

- Assédio, insultos ou ataques pessoais.
- Publicação de dados sensíveis.
- Manipulação deliberada de histórico ou decisões.
- Contribuições geradas por IA sem revisão mínima.
- Mudanças estruturais sem justificativa.

## Responsabilidades

Maintainers devem aplicar este código de forma consistente. Contributors devem seguir as regras de colaboração, segurança e governança do projeto.

## Próximos passos

- Alinhar este documento com a governança pública quando o projeto for publicado.
