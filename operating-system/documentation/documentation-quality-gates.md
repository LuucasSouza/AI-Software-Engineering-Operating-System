# Documentation Quality Gates

## Structure Gate

- [ ] Title exists.
- [ ] Purpose exists.
- [ ] Scope exists when relevant.
- [ ] Status and owner exist for canonical docs.
- [ ] Headings are coherent.

## Traceability Gate

- [ ] Related ADRs linked.
- [ ] Source artifacts linked.
- [ ] Dependencies identified.

## Usefulness Gate

- [ ] Document answers the reader's next question.
- [ ] Examples or checklists exist when action is expected.
- [ ] Vague statements are minimized.

## Consistency Gate

- [ ] Terminology matches AI-SEOS usage.
- [ ] File naming follows repository conventions.
- [ ] Indexes are updated.
