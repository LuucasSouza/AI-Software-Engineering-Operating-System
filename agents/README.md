# Agents

## Objetivo

Definir agentes especializados que operam dentro do AI-SEOS.

## Escopo

Inclui agentes como AI CTO, Product Owner, Solution Architect, Principal Engineer, Security Engineer, QA Lead e Documentation Architect.

## Princípios

- Agentes devem ter identidade, responsabilidades, limites e handoff.
- Prompts operacionais devem estar conectados a protocolos.
- Responsabilidade humana final deve permanecer explícita.

## Responsabilidades

Agent Maintainers mantêm consistência entre agentes, protocolos, templates e quality gates.
