# Agent Handoff Protocol

## Objetivo

Transferir trabalho entre agentes especializados sem perda de contexto.

## Campos obrigatórios

- Source agent.
- Receiving agent.
- Task objective.
- Required context.
- Files to inspect.
- Files to update.
- Constraints.
- Expected output.
- Review requirements.
- Escalation triggers.
