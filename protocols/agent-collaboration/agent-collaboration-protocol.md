# Agent Collaboration Protocol

## Flow

1. Select collaboration topology.
2. Assign agent operating contracts.
3. Define required artifacts.
4. Execute work by engine boundary.
5. Use handoff packages between agents.
6. Escalate conflicts to Decision Engine or human reviewer.
7. Capture outcomes in Documentation and Reflection.
