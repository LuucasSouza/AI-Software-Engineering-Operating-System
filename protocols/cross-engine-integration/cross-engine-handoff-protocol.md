# Cross-Engine Handoff Protocol

## Start Criteria

- Required upstream artifacts exist.
- Minimum quality gates are passed.
- Blocking conditions are absent or explicitly waived.
- Receiver responsibilities are clear.

## Flow

1. Identify source and target engine.
2. Validate required inputs.
3. Check quality gates.
4. Record blocking conditions.
5. Create handoff package.
6. Receiver accepts, rejects or requests rework.
7. Escalate unresolved gaps.
