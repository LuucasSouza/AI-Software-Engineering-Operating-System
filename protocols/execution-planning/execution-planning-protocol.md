# Execution Planning Protocol

## Objetivo

Converter artefatos upstream aprovados em execution plan, sprint plan, work packages e handoff packages.

## Fluxo

1. Coletar inputs.
2. Avaliar ERL.
3. Escolher delivery strategy.
4. Criar milestones.
5. Decompor work packages.
6. Mapear dependências.
7. Atribuir owners.
8. Definir quality gates.
9. Criar sprint plan.
10. Criar handoff packages.
11. Validar execution plan.

## Regra

Feature delivery não começa sem Execution Plan válido ou Spike Plan formal.
