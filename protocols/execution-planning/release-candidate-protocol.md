# Release Candidate Protocol

## Objetivo

Definir quando uma entrega pode ser tratada como release candidate.

## Critérios

- Work packages completos.
- Acceptance criteria satisfeitos.
- Testes executados ou waiver documentado.
- Documentação atualizada.
- Riscos conhecidos documentados.
- Rollback considerado.
- Handoff package preparado.
- Aprovação humana obtida quando exigida.
