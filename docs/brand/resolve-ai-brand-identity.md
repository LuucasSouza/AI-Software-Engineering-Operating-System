# Resolve Aí Brand Identity

## Public Name

Resolve Aí

## Official Promise

> Me dá o problema ou a ideia, e eu te ajudo a resolver.

## Relationship With AI-SEOS

AI-SEOS was the original technical and historical name of Resolve Aí. During migration, public-facing docs should use Resolve Aí, while historical ADRs, sprint reports and bootstrap-source files may keep AI-SEOS for traceability.

## Essence

Resolve Aí is practical, helpful, clear, approachable and technically serious underneath.

## What It Is

Resolve Aí is an open-source framework and future runtime layer that helps users turn problems, ideas and existing projects into structured software plans using AI.

## What It Is Not

Resolve Aí is not just a prompt pack, not just a README, not a replacement for human judgment and not only for professional engineers.

## Naming Rules

- Use `Resolve Aí` for public brand and human-facing docs.
- Use `resolve-ai` for future CLI, package and command references.
- Use `AI-SEOS` only for historical origin, technical lineage or legacy references.
