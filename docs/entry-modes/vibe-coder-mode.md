# Vibe Coder Mode

## Objetivo

Ajudar pessoas que constroem com ferramentas de IA a ganhar estrutura, guardrails, sequência e disciplina de engenharia.

## Usuário alvo

Pessoas que usam Codex, Cursor, Claude Code, Gemini CLI, Replit, Bolt, Lovable ou ferramentas similares e querem construir rápido sem criar caos.

## Princípio

Move fast, but with rails.

## Guardrails

- No giant implementation prompt.
- No stack choice without rationale.
- No database schema without domain understanding.
- No authentication shortcut for real users.
- No payment integration without risk review.
- No deploy without environment variable review.
- No feature expansion before MVP validation.

## Saída

Implementation phases, Codex-ready prompts, validation checkpoints, rollback notes and lightweight ADRs.
