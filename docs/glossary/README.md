# Glossary

## Objetivo

Definir termos canônicos usados no AI-SEOS para reduzir ambiguidade entre humanos e agentes.

## Escopo

Termos de engenharia, agentes, engines, protocolos, governança, handoff, ADRs e artefatos.

## Princípios

- Um termo deve ter uma definição preferida.
- Sinônimos devem apontar para o termo canônico.
- Definições devem ser práticas, não apenas acadêmicas.

## Responsabilidades

Maintainers devem atualizar o glossário quando novos conceitos forem introduzidos.
