# First Project with Resolve Ai

## Objetivo

Guiar o primeiro uso do Resolve Ai em um projeto real ou simulado.

## Passo 1 — Abra o projeto

Use um projeto sem dados sensiveis reais para o primeiro teste.

## Passo 2 — Inicie

```bash
resolve-ai começar
resolve-ai ligar
```

## Passo 3 — Diagnostique

```bash
resolve-ai diagnosticar
```

Leia:

```text
docs/resolve-ai/01-current-state-assessment.md
docs/resolve-ai/09-handoff.md
```

## Passo 4 — Planeje

```bash
resolve-ai planejar
```

Leia:

```text
docs/resolve-ai/10-plano-de-continuacao.md
docs/resolve-ai/11-backlog-priorizado.md
```

## Passo 5 — Prepare uma tarefa

```bash
resolve-ai preparar
resolve-ai resolver
```

O Resolve Ai ainda nao altera codigo. Ele prepara o contexto para voce ou seu agente.

## Passo 6 — Valide

Depois de executar a tarefa com revisao humana:

```bash
resolve-ai validar
resolve-ai status
```

## Observacao alpha

Esta versao e alpha. Use como apoio de organizacao, nao como piloto automatico.
