# Private Alpha Session Prep Checklist

## Environment

- [ ] Node version checked.
- [ ] npm version checked.
- [ ] CLI built locally.
- [ ] `resolve-ai ajuda` works.
- [ ] Temporary or sanitized project selected.
- [ ] No real secrets or customer data present.

## Session Flow

```text
ajuda -> começar -> ligar -> diagnosticar -> planejar -> preparar -> resolver -> validar -> status
```

## Moderator Reminders

- [ ] Observe before explaining too much.
- [ ] Do not defend the product.
- [ ] Record confusion as evidence.
- [ ] Stop if sensitive data appears.
- [ ] Confirm `resolver` is understood as assisted execution.
- [ ] Confirm `validar` is understood as guided review, not automatic test execution.
