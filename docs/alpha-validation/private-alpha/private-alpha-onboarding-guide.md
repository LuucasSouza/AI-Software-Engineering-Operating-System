# Private Alpha Onboarding Guide

## What Resolve Ai Is

Resolve Ai helps organize an idea, problem, or existing project into diagnosis, plan, prepared task, assisted execution package, and validation checklist.

## What Resolve Ai Does Not Do

- It does not alter code automatically.
- It does not publish packages.
- It does not create Git tags.
- It does not deploy.
- It does not collect telemetry.
- It does not call external APIs.

## Commands Used in the Session

```text
resolve-ai ajuda
resolve-ai começar
resolve-ai ligar
resolve-ai diagnosticar
resolve-ai planejar
resolve-ai preparar
resolve-ai resolver
resolve-ai validar
resolve-ai status
```

## Safety Note

Use only safe projects. Do not share credentials, customer data, `.env`, dumps, backups, or confidential repositories.

## Feedback Goal

The goal is not to prove the tool is ready. The goal is to find what is unclear, unsafe, excessive, or useful before public release.
