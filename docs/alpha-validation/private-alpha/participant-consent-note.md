# Private Alpha Participant Consent Note

## Simple Consent

```text
Voce esta participando de um teste privado do Resolve Ai.

A ferramenta esta em alpha, pode ter falhas e nao deve ser usada com dados sensiveis.

Seu feedback sera registrado de forma anonimizada para melhorar clareza, seguranca e utilidade do produto.

Voce pode parar a sessao a qualquer momento.
```

## Do Not Share

- Tokens.
- Senhas.
- `.env`.
- Dados de clientes.
- Dados pessoais.
- Dumps, backups ou credenciais.
- Codigo proprietario sem autorizacao.

## Allowed to Record

- Profile.
- General project type.
- Commands executed.
- Confusions observed.
- Feedback without identifying details.
- Scores and release recommendation.
