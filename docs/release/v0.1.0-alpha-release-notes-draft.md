# Resolve Ai v0.1.0-alpha.0 — Release Notes Draft

O Resolve Ai chegou em alpha para ajudar voce a transformar uma ideia ou projeto baguncado em diagnostico, plano, tarefa preparada, execucao assistida e validacao.

Ainda e alpha: ele nao mexe sozinho no seu codigo, nao publica nada, nao coleta telemetria e nao substitui seu julgamento.

## O que existe

- CLI local `resolve-ai`.
- Comandos em portugues.
- Modo Liga/Desliga.
- Diagnostico local seguro.
- Planejamento guiado.
- Preparo de tarefa.
- Execucao assistida com aprovacao humana.
- Validacao guiada pos-execucao.
- Stack provavel para Node.js, TypeScript, Vite, React, Firebase, Supabase, Python e monorepo simples.

## O que nao existe

- Publicacao npm nesta fase.
- Execucao automatica de tarefas.
- Telemetria.
- MCP.
- Hooks.
- Deploy, commit ou tag automaticos.
- Analise semantica profunda com IA.

## Recomendacao atual

LIMITED GO para alpha controlada. Public npm continua dependente de validacao real com usuarios.
