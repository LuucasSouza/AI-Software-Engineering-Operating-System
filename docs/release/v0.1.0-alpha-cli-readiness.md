# v0.1.0-alpha CLI Readiness

## Status

Alpha local candidate.

## O que existe

- CLI `resolve-ai` com comandos locais em português.
- Fluxo completo: `começar`, `ligar`, `diagnosticar`, `planejar`, `preparar`, `resolver`, `validar`, `status`, `desligar`.
- Build local por `npm run build --prefix packages/resolve-ai-cli`.
- Testes automatizados por `npm test --prefix packages/resolve-ai-cli`.
- Smoke test por `npm run smoke --prefix packages/resolve-ai-cli`.
- Documentação local em `docs/resolve-ai/`.
- Estado local em `.resolve-ai/`.

## O que não existe

- Publicação npm.
- MCP.
- Hooks.
- Telemetria.
- Chamadas externas.
- Execução automática de tarefas.
- Commit, push, tag ou deploy automáticos.
- Validação semântica profunda com IA.

- [x] Build funciona.

## Riscos conhecidos

- Diagnóstico e validação são heurísticos.
- `dist/` permanece versionado durante alpha.
- Build usa script local simples, não toolchain TypeScript completo.
- Estado local ainda pode mudar antes da versão estável.
