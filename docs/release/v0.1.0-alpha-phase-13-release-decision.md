# v0.1.0-alpha Phase 13 Release Decision

## Decisão

**LIMITED GO — alpha controlada, sem npm publish, sem tag e sem GitHub release.**

## Evidências

- Build, testes e smoke passam.
- `npm pack --dry-run` foi revisado dentro do pacote.
- `package.json` agora usa `files` para reduzir conteúdo publicado.
- O dry-run caiu para 32 arquivos, sem `src/` e `tests/`.
- Fluxo local mínimo passa em diretório temporário.
- P1/P2 técnicos das fases anteriores foram corrigidos ou mitigados.

## Bloqueio para GO público

Nenhuma validação real com usuários foi executada nesta fase.

## O que fica permitido

- Continuar validação controlada.
- Compartilhar instruções locais com usuários convidados.
- Coletar feedback manual usando templates.

## O que continua proibido

- `npm publish`.
- `git tag`.
- GitHub release.
- Declarar stable.

## Critério para GO futuro

Registrar validação real mínima e revisar blockers P1 antes de executar release.
