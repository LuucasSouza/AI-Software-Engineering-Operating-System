# v0.1.0-alpha Known Limitations

- Extensive docs remain draft in areas.
- Templates need more real-world validation.
- Link validation is manual.
- Metadata consistency needs automation.
- CLI exists locally in `packages/resolve-ai-cli/`, but it is not published on npm.
- No stable schema for `.resolve-ai/state.json`.
- Stack detection/exhibition needs improvement for Node and Vite/React before public npm release.
- External validation with real users is still pending.
- Examples are documentation artifacts, not complete runnable products.
