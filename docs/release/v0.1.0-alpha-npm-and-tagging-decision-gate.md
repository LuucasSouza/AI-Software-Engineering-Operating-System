# v0.1.0-alpha npm and Tagging Decision Gate

## Acoes proibidas nesta fase

- `npm publish`
- `git tag`
- `git push --tags`
- GitHub release

## Acoes permitidas

- `npm pack --dry-run --prefix packages/resolve-ai-cli`
- build, tests e smoke locais
- revisao manual de package contents

## Decisao recomendada

**LIMITED GO**.

Pode preparar release execution, mas publicacao npm e tag devem permanecer manuais e condicionadas a validacao real.

## Evidencias necessarias para GO futuro

- Sem P0/P1 bloqueante.
- Validacao real minima registrada.
- `npm pack --dry-run` revisado.
- Release notes aprovadas.
- Confirmacao manual do mantenedor.
