# v0.1.0-alpha Phase 15A Release Recommendation

## Recommendation

Continue private alpha. Do not publish npm.

## Recommended Next Phase

Phase 15B - Private Alpha Fixes and Second Validation Round.

## Why

The CLI is technically healthy, but no real participant validated the experience in Phase 15A. Public alpha needs evidence from Non-Technical Builder, Vibe Coder, and Professional Engineer profiles.

## Allowed

- Invite private alpha participants.
- Run safe local sessions.
- Collect anonymized feedback.
- Apply small evidence-based UX/docs fixes.

## Not Allowed

- npm publish.
- Git tag.
- GitHub release.
- Stable claim.
