Score: **79/100**.

Status recomendado: **LIMITED GO — alpha controlada, sem npm public alpha ainda**.

## Blockers P0

Nenhum P0 foi observado nos cenarios simulados executados.

## P1 Restantes

- Validacao com usuarios reais ainda nao foi executada.
- Aprovação manual do mantenedor para npm/tag ainda pendente.

## P1 Corrigidos ou Mitigados

- Deteccao/exibicao de stack para Node.js, TypeScript, Vite e React corrigida em fixtures locais.
- Heuristicas simples adicionadas para Firebase, Supabase, Python e monorepo.
- Copy do `resolver` atualizada para indicar que `resolve-ai validar` ja existe.
- Hardening em projeto novo calibrado para nao virar critico por ausencia isolada de testes/CI.
- Package dry-run revisado e `files` aplicado para reduzir tarball.

## Principios

- O score reflete evidencia local simulada.
- A ausencia de usuarios externos reais reduz readiness.
- Qualquer P0 futuro deve bloquear publicacao mesmo se o score subir.

## Responsabilidades

Maintainers devem atualizar este score apos cada rodada externa.

## Proximos passos

Rodar validacao controlada com usuarios reais antes de npm public alpha.
