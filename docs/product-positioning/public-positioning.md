# Public Positioning

## Official Name

Resolve Aí

## Promise

> Me dá o problema ou a ideia, e eu te ajudo a resolver.

## Short Description

Resolve Aí helps you turn an idea, a problem or an existing project into a clear software plan using AI.

## Public Explanation

Resolve Aí helps you organize an idea before asking AI to code it. It supports discovery, product definition, architecture, decisions, risks, execution planning, documentation and handoff.

## Audience Paths

- I have an idea or problem, but I do not code: Non-Technical Builder.
- I use AI to code, but my projects get messy: Vibe Coder.
- I am a developer or technical team: Professional Engineer.

## Former Name

Resolve Aí was originally created under the technical name AI-SEOS, short for AI Software Engineering Operating System.
