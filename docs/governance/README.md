# Governance Docs

## Objetivo

Agrupar documentos de governança que apoiam evolução, revisão e manutenção do AI-SEOS.

## Escopo

Inclui políticas, processos de decisão, papéis, revisão, compatibilidade e evolução do projeto.

## Princípios

- Decisões estruturais exigem ADR.
- Mudanças devem ser rastreáveis.
- Governança deve proteger qualidade sem travar evolução.

## Responsabilidades

Governance Maintainers mantêm alinhamento com `GOVERNANCE.md` e ADRs aceitas.
